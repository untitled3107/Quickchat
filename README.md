# QuickChat 💬

QuickChat is a real-time chat application built with the MERN stack (MongoDB, Express, React, and Node.js), designed to deliver fast, seamless one-on-one messaging. The app allows users to sign up, log in securely, and chat instantly with other users through a clean and responsive interface.

Built with a focus on real-time communication, QuickChat uses modern web technologies to ensure messages are delivered instantly without delays. The UI is styled using Tailwind CSS, offering a sleek, minimal design with support for profile pictures, timestamps, and message bubbles that distinguish between sent and received messages.

On the backend, the application handles user authentication securely using JWT, while MongoDB stores user data and chat history efficiently. The architecture separates concerns between the frontend and backend, making the codebase scalable and easy to maintain.

This project was built as a hands-on way to explore full-stack development, real-time data handling, and authentication flows — combining practical UI/UX design with backend logic to create a smooth chatting experience for users.
